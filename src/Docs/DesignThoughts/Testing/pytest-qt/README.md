# Testing with pytest-qt

Open source library for testing PySide6, PyQt6 and PyQt5.

Can test projects in a headless manner.

# References:

[pytest-qt](https://github.com/pytest-dev/pytest-qt) github project

[pytest-qt](https://pytest-qt.readthedocs.io/en/latest/intro.html) documentation

[Headless Testing of PySide/PyQt GUI Applications with pytest-qt](https://ilmanzo.github.io/post/testing_pyside_gui_applications/)
